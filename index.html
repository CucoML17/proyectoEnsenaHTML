Aprendamos HTML y sus etiquetas más importantes

Las páginas web son herramientas de divulgación de información con múltiples fines: desde brindar conocimiento, hasta generar oportunidades de negocio.

HTML son las siglas en inglés para Hypertext Markup Language, que se traduce como lenguaje de marcación de hipertexto. es por ello que se le considera la piedra angular del desarrollo web, y un buen dominio de este lenguaje es esencial para crear y mantener sitios web efectivos y funcionales.

¿Qué es HTML?

HTML, o Lenguaje de Marcado de Hipertexto, es un lenguaje de programación que se utiliza para definir la estructura y el contenido de las páginas web; permite crear páginas web estáticas con elementos como texto, encabezados, tablas, listas, imágenes y enlaces.

El HTML es el componente más básico de la web y se compone de una serie de etiquetas y elementos que le indican al navegador cómo mostrar la información a los visitantes.

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.



Estructura del HTML

La estructura HTML de una página web se compone de las etiquetas html, que indica el comienzo de la página; head, que contiene información más descriptiva sobre el sitio; y body, que conforma todos los elementos visibles de la página.

¿Qué es Fronted?

El frontend o desarrollo del lado del cliente se refiere a la práctica de producir HTML, CSS y JavaScript. Estos tres elementos se encargan de dar forma a la parte frontal de un sitio web o aplicación. Esto incluye los fondos, colores, texto, animaciones o efectos.

Precisamente de ahí proviene el nombre de desarrollo del lado del cliente, pues con el frontend se puede construir por completo lo que los usuarios perciben al explorar un sitio y con el que pueden interactuar.


¿Qué es Backend?

El backend es el encargado de procesar toda la información que alimenta a un frontend. Se compone de marcos, bases de datos o códigos. Para que un sitio web o aplicación opere efectivamente, se requiere mucha información y datos que se almacenan en la parte trasera de un sistema informático. En oposición al frontend, el usuario no puede ver o acceder a esta información.

El backend son todos los códigos ocultos que sirven para que una página web o aplicación funcione correctamente. Además, de su estructura y organización depende la experiencia de usuario. De igual forma, el backend se encarga de optimizar otros elementos y recursos como la seguridad y privacidad en un sitio web o aplicación.



¿Qué es CSS?

CSS son las siglas de Cascading Style Sheets, que en español se traduce como Hojas de Estilo en Cascada. Se trata de un lenguaje de programación que permite estructurar y componer páginas web, como las que están hechas en HTML o XML.

El CSS se encarga de seleccionar elementos de una página web y controlar su presentación. Para ello, se basa en reglas que el usuario define para aplicar estilos a elementos específicos o grupos de elementos. Por ejemplo, se puede indicar que el encabezado principal de una página se muestre en letras grandes y de color rojo.



¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas. Es una de las tres tecnologías web principales, junto con HTML y CSS, y se utiliza para ofrecer experiencias web dinámicas.

JavaScript se puede utilizar para: Actualizar fuentes de redes sociales, Mostrar animaciones y mapas interactivos, Controlar multimedia, Crear juegos, Construir servidores web y aplicaciones de servidor.



¿Qué es PHP?

PHP es un lenguaje de programación de código abierto, gratuito y del lado del servidor, que se usa principalmente para crear sitios web dinámicos.

PHP es un acrónimo de Hypertext Preprocessor, que significa Preprocesador de Hipertexto. Fue creado en 1994 por el programador Rasmus Lerdorf y actualmente es implementado por The PHP Group.
