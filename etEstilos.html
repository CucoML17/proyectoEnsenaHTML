Estilos principales de CSS

Los atributos en HTML proporcionan información adicional sobre un elemento HTML. Se colocan dentro de las etiquetas de los elementos y están compuestos por un nombre y un valor.
Los atributos modifican el comportamiento y la apariencia de los elementos HTML.

CSS (Cascading Style Sheets) se utiliza para definir la presentación y el diseño de los elementos HTML. Los estilos CSS permiten controlar aspectos visuales como colores, fuentes, márgenes y posiciones.

Atributo color

color

El atributo color en CSS se utiliza para especificar el color del texto de un elemento HTML. Permite cambiar el color de cualquier texto dentro del elemento al que se aplique.

La propiedad color define el color del texto en el contenido de un elemento HTML. Puede afectar a etiquetas como "p", "h1", "div", y cualquier otro elemento que contenga texto. Además, se puede usar para definir colores en pseudo-elementos como ::before y ::after.

Atributos

  • Nombres de colores: Utiliza nombres predefinidos en CSS. Ejemplos incluyen:
    • red - Rojo
    • blue - Azul
    • green - Verde
    • black - Negro
    • white - Blanco
  • Código hexadecimal: Representa colores usando un código de 6 dígitos precedido por #. Ejemplos:
    • #FF5733 - Un tono de naranja
    • #00FF00 - Verde puro
  • Valores RGB: Especifica colores usando tres valores numéricos para rojo, verde y azul en el formato rgb(255, 0, 0). Ejemplos:
    • rgb(255, 0, 0) - Rojo puro
    • rgb(0, 255, 0) - Verde puro
  • Valores RGBA: Similar a RGB pero con un valor adicional para la opacidad (alpha), en el formato rgba(255, 0, 0, 0.5). Ejemplos:
    • rgba(255, 0, 0, 0.5) - Rojo con 50% de opacidad
    • rgba(0, 255, 0, 0.75) - Verde con 75% de opacidad
  • Valores HSL: Representa colores usando tono, saturación y luminosidad en el formato hsl(120, 100%, 50%). Ejemplos:
    • hsl(120, 100%, 50%) - Verde puro
    • hsl(240, 100%, 50%) - Azul puro
  • Valores HSLA: Similar a HSL pero con un valor adicional para la opacidad, en el formato hsla(120, 100%, 50%, 0.5). Ejemplos:
    • hsla(120, 100%, 50%, 0.5) - Verde con 50% de opacidad
    • hsla(240, 100%, 50%, 0.75) - Azul con 75% de opacidad

Atributo font-size

font-size

La propiedad font-size en CSS se utiliza para especificar el tamaño de la fuente del texto de un elemento HTML. Permite ajustar el tamaño del texto dentro del elemento al que se aplique.

La propiedad font-size define el tamaño de la fuente en el contenido de un elemento HTML. Puede afectar a etiquetas como "p", "h1", "div", y cualquier otro elemento que contenga texto. Además, se puede usar para ajustar el tamaño de texto en pseudo-elementos como ::before y ::after.

Atributos

  • Tamaño en píxeles (px): Define el tamaño de la fuente en píxeles. Ejemplos:
    • font-size: 16px; - Tamaño de fuente de 16 píxeles
    • font-size: 24px; - Tamaño de fuente de 24 píxeles
  • Tamaño en em: Define el tamaño de la fuente en relación al tamaño de la fuente del elemento padre. Ejemplos:
    • font-size: 1em; - Tamaño de fuente igual al tamaño del elemento padre
    • font-size: 1.5em; - Tamaño de fuente 1.5 veces el tamaño del elemento padre
  • Tamaño en rem: Define el tamaño de la fuente en relación al tamaño de la fuente raíz (generalmente el tamaño de la fuente del elemento html). Ejemplos:
    • font-size: 1rem; - Tamaño de fuente igual al tamaño de la fuente raíz
    • font-size: 2rem; - Tamaño de fuente 2 veces el tamaño de la fuente raíz
  • Tamaño en porcentaje (%): Define el tamaño de la fuente en porcentaje del tamaño de la fuente del elemento padre. Ejemplos:
    • font-size: 100%; - Tamaño de fuente igual al tamaño del elemento padre
    • font-size: 150%; - Tamaño de fuente 1.5 veces el tamaño del elemento padre
  • Tamaño en puntos (pt): Define el tamaño de la fuente en puntos. Ejemplos:
    • font-size: 12pt; - Tamaño de fuente de 12 puntos
    • font-size: 18pt; - Tamaño de fuente de 18 puntos

Atributo margin

margin

La propiedad margin en CSS se utiliza para especificar el espacio exterior (margen) alrededor de un elemento HTML. Controla la separación entre el elemento y otros elementos adyacentes en la página.

La propiedad margin define los márgenes de un elemento y puede aplicar márgenes uniformes en los cuatro lados o márgenes específicos para cada lado: superior, inferior, izquierdo y derecho. También puede aceptar valores en píxeles, porcentajes, o unidades relativas como em/rem.

Atributos

  • Valor único: Aplica el mismo margen a los cuatro lados del elemento. Ejemplo:
    • margin: 20px; - Aplica un margen de 20 píxeles en todos los lados
  • Cuatro valores: Especifica márgenes diferentes para cada lado en el orden: superior, derecho, inferior, izquierdo. Ejemplo:
    • margin: 10px 20px 30px 40px; - Aplica 10px arriba, 20px a la derecha, 30px abajo y 40px a la izquierda
  • Tres valores: El primer valor es para el margen superior, el segundo para los márgenes derecho e izquierdo, y el tercero para el margen inferior. Ejemplo:
    • margin: 10px 20px 30px; - Aplica 10px arriba, 20px a los lados y 30px abajo
  • Dos valores: El primer valor es para los márgenes superior e inferior, y el segundo para los márgenes derecho e izquierdo. Ejemplo:
    • margin: 10px 20px; - Aplica 10px arriba y abajo, y 20px a los lados
  • Valor auto: Puede utilizarse para centrar un elemento horizontalmente dentro de su contenedor. Ejemplo:
    • margin: 0 auto; - Aplica 0 de margen arriba y abajo, y centra el elemento horizontalmente
  • Valores negativos: Se pueden aplicar márgenes negativos para solapar elementos. Ejemplo:
    • margin: -10px; - Aplica un margen negativo de 10 píxeles, lo que hace que el elemento se superponga a otros

Atributo padding

Padding

La propiedad padding en CSS se utiliza para especificar el espacio interior (relleno) entre el contenido de un elemento y su borde. Controla la cantidad de espacio entre el borde del elemento y su contenido, afectando directamente al área visible dentro del elemento.

La propiedad padding define el relleno de un elemento y puede aplicar valores uniformes en los cuatro lados o especificar valores para cada lado: superior, inferior, izquierdo y derecho. Acepta valores en unidades como píxeles, porcentajes, o unidades relativas como em/rem.

Atributos

  • Valor único: Aplica el mismo relleno a los cuatro lados del elemento. Ejemplo:
    • padding: 20px; - Aplica un relleno de 20 píxeles en todos los lados
  • Cuatro valores: Especifica rellenos diferentes para cada lado en el orden: superior, derecho, inferior, izquierdo. Ejemplo:
    • padding: 10px 20px 30px 40px; - Aplica 10px arriba, 20px a la derecha, 30px abajo y 40px a la izquierda
  • Tres valores: El primer valor es para el relleno superior, el segundo para los rellenos derecho e izquierdo, y el tercero para el relleno inferior. Ejemplo:
    • padding: 10px 20px 30px; - Aplica 10px arriba, 20px a los lados y 30px abajo
  • Dos valores: El primer valor es para los rellenos superior e inferior, y el segundo para los rellenos derecho e izquierdo. Ejemplo:
    • padding: 10px 20px; - Aplica 10px arriba y abajo, y 20px a los lados
  • Valores en porcentaje (%): El relleno se puede definir en porcentaje del ancho del elemento padre. Ejemplo:
    • padding: 5%; - Aplica un relleno del 5% del ancho del contenedor

Atributo border

border

La propiedad border en CSS se utiliza para definir el borde alrededor de un elemento HTML. Esta propiedad permite especificar el estilo, el grosor y el color del borde que rodea el contenido y el relleno de un elemento.

La propiedad border combina tres aspectos principales: el grosor (width), el estilo (style) y el color (color) del borde. Puede aplicarse de forma global para todos los lados del elemento o individualmente para cada lado (superior, inferior, izquierdo y derecho).

Atributos

  • Grosor del borde: Define el grosor del borde. Puede ser en píxeles, ems o palabras clave predefinidas como thin, medium, o thick. Ejemplo:
    • border: 2px solid black; - Borde de 2 píxeles de grosor, sólido, color negro
  • Estilo del borde: Especifica el estilo del borde. Valores comunes incluyen:
    • solid - Un borde sólido
    • dashed - Un borde discontinuo
    • dotted - Un borde punteado
    • double - Un borde doble
    • none - Sin borde
  • Color del borde: Define el color del borde. Puede ser cualquier valor de color aceptado por CSS. Ejemplo:
    • border: 2px solid red; - Borde rojo de 2 píxeles de grosor
  • Border shorthand: La propiedad border permite combinar grosor, estilo y color en una sola línea. Ejemplo:
    • border: 1px dashed blue; - Borde azul, discontinuo, de 1 píxel de grosor
  • Bordes específicos: Puedes definir bordes para lados específicos del elemento:
    • border-top, border-right, border-bottom, border-left - Define el borde en un lado específico
    • border-left: 3px solid green; - Borde izquierdo de 3 píxeles, sólido, color verde

Atributo width

width

La propiedad width en CSS se utiliza para especificar el ancho de un elemento. Esta propiedad controla la anchura del área de contenido de un elemento, excluyendo los márgenes, rellenos y bordes.

La propiedad width define el ancho de un elemento y puede aceptar valores absolutos (como píxeles) o relativos (como porcentajes). Se puede aplicar a casi cualquier elemento HTML que tenga dimensiones definibles, como contenedores (div, section, aside, etc.).

Atributos

  • Valor en píxeles: Define el ancho en píxeles. Ejemplo:
    • width: 300px; - El ancho del elemento será de 300 píxeles
  • Valor en porcentaje: Define el ancho en relación con el ancho del elemento contenedor. Ejemplo:
    • width: 50%; - El ancho del elemento será el 50% del contenedor padre
  • Valores automáticos: Deja que el navegador calcule automáticamente el ancho del elemento. Ejemplo:
    • width: auto; - El ancho será determinado automáticamente por el contenido y el entorno del elemento
  • Valores mínimos y máximos: Puedes usar las propiedades min-width y max-width para limitar el tamaño mínimo o máximo que puede tener un elemento:
    • min-width: 200px; - El ancho mínimo será de 200 píxeles
    • max-width: 600px; - El ancho máximo será de 600 píxeles

Atributo height

height

La propiedad height en CSS se utiliza para especificar la altura de un elemento. Esta propiedad controla la altura del área de contenido de un elemento, excluyendo los márgenes, rellenos y bordes.

La propiedad height define la altura de un elemento y puede aceptar valores absolutos (como píxeles) o relativos (como porcentajes). Al igual que width, se puede aplicar a cualquier elemento HTML con dimensiones, como cajas de contenedores (div, section, etc.).

Atributos

  • Valor en píxeles: Define la altura en píxeles. Ejemplo:
    • height: 200px; - La altura del elemento será de 200 píxeles
  • Valor en porcentaje: Define la altura en relación con el alto del elemento contenedor. Ejemplo:
    • height: 75%; - La altura del elemento será el 75% del contenedor padre
  • Valores automáticos: Permite que el navegador ajuste automáticamente la altura del elemento según su contenido. Ejemplo:
    • height: auto; - La altura será ajustada automáticamente
  • Valores mínimos y máximos: Puedes usar min-height y max-height para establecer límites en la altura:
    • min-height: 100px; - La altura mínima será de 100 píxeles
    • max-height: 500px; - La altura máxima será de 500 píxeles

Atributo text-align

Text-align

La propiedad text-align en CSS se utiliza para especificar la alineación horizontal del texto dentro de un elemento. Controla cómo se alinean las líneas de texto y los elementos en línea dentro de un contenedor.

La propiedad text-align se puede aplicar a cualquier contenedor de bloque, como un div, section, o p. Afecta principalmente a elementos de texto y también puede influir en otros elementos en línea como imágenes o enlaces.

Atributos

  • left: Alinea el texto a la izquierda del contenedor. Ejemplo:
    • text-align: left; - El texto estará alineado a la izquierda
  • right: Alinea el texto a la derecha del contenedor. Ejemplo:
    • text-align: right; - El texto estará alineado a la derecha
  • center: Centra el texto dentro del contenedor. Ejemplo:
    • text-align: center; - El texto estará centrado
  • justify: Justifica el texto, distribuyendo el espacio entre palabras para que las líneas tengan la misma longitud. Ejemplo:
    • text-align: justify; - El texto estará justificado
  • start: Alinea el texto al comienzo según la dirección del texto. Ejemplo:
    • text-align: start; - Alinea el texto al inicio según la dirección (izquierda para idiomas como el español e inglés)
  • end: Alinea el texto al final según la dirección del texto. Ejemplo:
    • text-align: end; - Alinea el texto al final según la dirección (derecha para idiomas como el español e inglés)

Atributo display

<div> </div>

La propiedad display en CSS determina cómo se va a mostrar un elemento en la página. Controla si el elemento se comporta como un bloque, un elemento en línea, o si debe ocultarse del diseño visual sin eliminarse del árbol del DOM.

Esta propiedad es fundamental para manipular la estructura de una página, ya que puede cambiar completamente el flujo de elementos en el diseño. Se aplica a casi cualquier elemento HTML y se usa ampliamente en diseños de página web.

Atributos

  • block: Hace que el elemento se comporte como un bloque. Ocupa todo el ancho disponible y se coloca en una nueva línea. Ejemplo:
    • display: block; - El elemento será un bloque
  • inline: Hace que el elemento se comporte como un elemento en línea. No inicia una nueva línea y solo ocupa el espacio necesario. Ejemplo:
    • display: inline; - El elemento será en línea
  • inline-block: Combina características de elementos en línea y de bloque. Permite definir el ancho y alto pero no inicia una nueva línea. Ejemplo:
    • display: inline-block; - El elemento será en línea pero con las propiedades de un bloque
  • none: Oculta completamente el elemento. No será visible y no ocupará espacio en el diseño. Ejemplo:
    • display: none; - El elemento no aparecerá en el documento
  • flex: Aplica un contenedor flexible que ajusta el contenido para optimizar el espacio dentro del contenedor. Ejemplo:
    • display: flex; - El contenedor se convertirá en un contenedor flexible
  • grid: Crea un contenedor de cuadrícula que permite organizar elementos en filas y columnas. Ejemplo:
    • display: grid; - El contenedor se organizará en una cuadrícula
