Etiquetas para agrupar contenido

Las etiquetas de agrupación de contenido en HTML son elementos que permiten organizar y estructurar el contenido de una página web, facilitando su legibilidad y presentación. Estas etiquetas son fundamentales para crear una jerarquía visual y semántica en el documento HTML.

Etiqueta <p>

<p> </p>

Esta etiqueta se utiliza para definir un párrafo de texto. Es una de las etiquetas más comunes y básicas en la creación de contenido web.

Su función principal es agrupar oraciones que están relacionadas entre sí, lo que ayuda a mejorar la legibilidad y la estructura del contenido.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el párrafo con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del párrafo.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el párrafo, ofreciendo información adicional.
  • lang: Define el idioma del contenido dentro del párrafo, útil para accesibilidad y motores de búsqueda.
  • dir: Especifica la dirección del texto dentro del párrafo, como de izquierda a derecha (ltr) o de derecha a izquierda (rtl).
  • hidden: Oculta el párrafo del renderizado visual y de las tecnologías de asistencia.
  • data-*: Permite almacenar datos personalizados en el párrafo para su uso en JavaScript.
  • tabindex: Establece el orden de tabulación del párrafo cuando se navega con el teclado.
  • contenteditable: Especifica si el contenido del párrafo es editable por el usuario.

Ejemplo


Línea 9: id="intro" Asigna el identificador único 'intro' al párrafo, permitiendo su selección específica en CSS o JavaScript.
Línea 10: class="highlight" Aplica la clase 'highlight' al párrafo, permitiendo la aplicación de estilos CSS definidos en la hoja de estilos.
Línea 11: 'style="color: blue; font-size: 18px;"' Aplica estilos en línea al párrafo, configurando el color del texto en azul y el tamaño de la fuente en 18 píxeles.
Línea 13: 'title="Este es un párrafo de introducción"' Proporciona un *tooltip* que muestra "Este es un párrafo de introducción" cuando el usuario pasa el cursor sobre el párrafo.
Línea 14: 'lang="es"' Especifica que el contenido del párrafo está en español, mejorando la accesibilidad y el SEO.
Línea 15: 'dir="ltr"' Indica que el texto dentro del párrafo se debe leer de izquierda a derecha, que es la dirección de texto predeterminada en la mayoría de los casos.
Línea 16: 'contenteditable="true"' Hace que el contenido del párrafo sea editable directamente en el navegador por el usuario.



Etiqueta <hr>

<hr> </hr>

Esta etiqueta en HTML representa una línea horizontal que se utiliza para separar contenido temático o secciones dentro de una página web.

Estas etiquetas son fundamentales para crear una jerarquía visual y semántica en el documento HTML.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el `hr´ con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia de la línea horizontal, como su color, grosor, y estilo de borde.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre la línea horizontal, ofreciendo información adicional.
  • width: Define el ancho de la línea horizontal, ya sea en píxeles o porcentaje relativo al contenedor.
  • size: Controla el grosor de la línea horizontal.
  • align: Alinea la línea horizontal dentro de su contenedor (atributo obsoleto en HTML5, mejor usar CSS).
  • noshade: Aplica una línea sin sombreado, creando una línea sólida (atributo obsoleto, mejor usar CSS).

Etiqueta <pre>

<pre> </pre>

La etiqueta "pre" se utiliza para definir texto preformateado, lo que significa que el texto dentro de esta etiqueta se mostrará con espacios y saltos de línea tal como se escribieron en el código fuente.

Es especialmente útil para mostrar código, poesía o cualquier contenido donde la presentación del texto sea importante.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el contenido del `pre` con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del contenido del `pre`, como color de texto, fuentes y márgenes.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el contenido del `pre`, ofreciendo información adicional.
  • lang: Especifica el idioma del contenido dentro del `pre`, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del `pre`, como de izquierda a derecha (ltr) o de derecha a izquierda (rtl).
  • hidden: Oculta el contenido del `pre` del renderizado visual y de las tecnologías de asistencia.
  • data-*: Permite almacenar datos personalizados en el `pre` para su uso en JavaScript.
  • tabindex: Establece el orden de tabulación del `pre` cuando se navega con el teclado.
  • contenteditable: Especifica si el contenido del `pre` es editable por el usuario.

Ejemplo

Línea 21: class="codigo-ejemplo" La clase 'codigo-ejemplo' se aplica al bloque ´pre´, permitiendo el uso de estilos CSS definidos en la hoja de estilos.
Línea 22: style="font-size: 16px;" Estilo en línea que establece el tamaño de la fuente en 16 píxeles.
Línea 23: title="Código de ejemplo" El texto emergente (tooltip) muestra "Código de ejemplo" al pasar el cursor sobre el bloque ´pre´.
Línea 24: lang="en" Indica que el contenido está en inglés.
Línea 25: dir="ltr" Define la dirección del texto como de izquierda a derecha.
Línea 26: hidden Oculta el bloque ´pre´ del renderizado visual y de tecnologías de asistencia.
Línea 27: data-author="Pelaez" Almacena el dato personalizado 'author' con el valor "Pelaez", accesible a través de JavaScript.
Línea 28: tabindex="0" Permite que el bloque ´pre´ sea parte del orden de tabulación en la navegación con el teclado.
Línea 29: contenteditable="true" Permite que el contenido del ´pre´ sea editable directamente en el navegador por el usuario.

  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el contenido del blockquote, ofreciendo información adicional.
  • lang: Especifica el idioma del contenido dentro del blockquote, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del blockquote, como de izquierda a derecha (ltr) o de derecha a izquierda (rtl).
  • hidden: Oculta el contenido del blockquote del renderizado visual y de las tecnologías de asistencia.
  • data-*: Permite almacenar datos personalizados en el blockquote para su uso en JavaScript.
  • tabindex: Define el orden de tabulación del blockquote en la navegación con el teclado.

Etiqueta <ol>

<ol> </ol>

Se utiliza para definir una lista ordenada, donde los elementos de la lista se presentan en un orden específico, generalmente numerados.

Esta etiqueta es útil para mostrar secuencias, pasos o cualquier contenido que requiera un orden lógico.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el contenido del ´ol´ con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del ´ol´, como márgenes, color de fondo, o fuente.
  • type: Define el tipo de marcador utilizado para los números en la lista ordenada. Los valores posibles son "1" (números arábigos), "A" (letras mayúsculas), "a" (letras minúsculas), "I" (números romanos en mayúsculas), y "i" (números romanos en minúsculas).
  • start: Especifica el número inicial de la lista ordenada. Permite comenzar la numeración desde un valor diferente al predeterminado (1).
  • reversed: Indica que la lista debe ser ordenada en orden descendente en lugar de ascendente. Este atributo es útil cuando se desea mostrar los elementos en orden inverso.
  • lang: Especifica el idioma del contenido dentro del ´ol´, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del ´ol´, como de izquierda a derecha (ltr) o de derecha a izquierda (rtl).

Ejemplo




Línea 16: class="lista-importante": Aplica la clase lista-importante a la lista, permitiendo aplicar estilos CSS definidos en la hoja de estilos.
Línea 17: id="lista1": Asigna el identificador único lista1 a la lista, útil para referenciarla en CSS o JavaScript de manera específica.
Línea 18: style="font-size: 18px; margin: 20px;": Aplica estilos en línea, configurando el tamaño de la fuente en 18 píxeles y el margen en 20 píxeles.
Línea 19: type="A": Establece que la lista use letras mayúsculas (A, B, C, ...) para los números de los ítems.
Línea 20: start="3": Inicia la numeración de la lista desde el número 3.
Línea 21: reversed: Muestra la lista en orden descendente, por lo que el último ítem en el código será el primero mostrado.
Línea 22: lang="es": Define que el contenido de la lista está en español.
Línea 23: dir="ltr": Establece que la dirección del texto es de izquierda a derecha.

Etiqueta <figure>

<figure> </figure>

Se utiliza para encapsular contenido independiente, como imágenes, ilustraciones, diagramas, o fragmentos de código, que se relacionan con el texto principal de un documento.

Generalmente, se acompaña de una etiqueta figcaption para proporcionar una leyenda o descripción del contenido.

Atributos

  • class: Asigna una o más clases al elemento figure, permitiendo aplicar estilos CSS específicos o seleccionar el elemento con JavaScript.

Etiqueta <div>

<div> </div>

Es un elemento de nivel de bloque que se utiliza para agrupar y contener otros elementos HTML. A diferencia de otras etiquetas, div no tiene un significado semántico específico.

Atributos

  • class: Asigna una o más clases al elemento <div>, permitiendo aplicar estilos CSS específicos o seleccionar el elemento con JavaScript.
  • id: Proporciona un identificador único al elemento <div>, útil para aplicar estilos específicos o manipular el contenido con JavaScript.

  • hidden: Oculta el contenido del elemento <div> del renderizado visual sin eliminarlo del DOM.
  • data-*: Permite almacenar datos personalizados en el elemento <div>, accesibles mediante JavaScript usando el prefijo "data-" seguido de un nombre personalizado.
