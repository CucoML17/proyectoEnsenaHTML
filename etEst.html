Etiquetas de estructura

Las etiquetas de estructura en HTML se refieren a un conjunto de elementos que se utilizan para organizar y definir la estructura de un documento HTML. Estas etiquetas son fundamentales para mejorar la semántica, la accesibilidad y la legibilidad del código.

Etiqueta <body>

<body> </body>

Se utiliza para encapsular todo el contenido visible de un documento web, incluyendo texto, imágenes, enlaces, y otros elementos multimedia.

Atributos

  • class: Asigna una o más clases al ``, permitiendo aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al ``, útil para aplicar estilos específicos o manipular el elemento con JavaScript.
  • style: Permite aplicar estilos CSS en línea al ``, personalizando su apariencia.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el ``, ofreciendo información adicional.
  • data-*: Permite almacenar datos personalizados en el `` para su uso en JavaScript.
  • role: Define el papel o propósito del `` en la página, ayudando con la accesibilidad.

Ejemplo


Línea 12:id="mainBody" Asigna un identificador único 'mainBody' al `body`, útil para aplicar estilos específicos o manipularlo con JavaScript.
Línea 13: class="background" Asigna la clase 'background' al `body`, permitiendo aplicar el estilo definido en la hoja de estilos.
Línea 14: style="margin: 0;" Aplica un margen de 0 píxeles en línea al `body`.
Línea 15: title="Cuerpo del documento" Proporciona un *tooltip* que muestra "Cuerpo del documento" cuando el usuario pasa el cursor sobre el `body`.
Línea 16: data-custom="value" Almacena un dato personalizado 'value' en el `body` para su uso en JavaScript.
Línea 17: role="document" Define el papel del `body` como el documento en la página, ayudando con la accesibilidad.



Etiqueta <nav>

<nav> </nav>

Se utiliza para definir un bloque de navegación que contiene enlaces a otras páginas o secciones dentro del mismo documento. Es fundamental para la estructura de la navegación en un sitio web, mejorando la accesibilidad y la usabilidad.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el elemento con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del elemento `nav`.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el elemento, ofreciendo información adicional.

Ejemplo



Línea 9: `class="menu"` Asigna la clase `menu` al `nav`, permitiendo aplicar estilos CSS a toda la sección de navegación que usa esta clase. Los estilos definidos para `.menu` en la hoja de estilos se aplicarán a este `nav`.
Línea 11, 12, 13: `href="#"` El atributo `href` define el destino del enlace. En estos casos, `#` actúa como un marcador de posición. En una aplicación real, `href` debería ser una URL válida o un identificador de anclaje para navegar a diferentes secciones de la misma página o a otras páginas.

Etiqueta <main>

<main> </main>

Se utiliza para definir el contenido principal de un documento. Este contenido es relevante y central para el propósito del documento, y debe ser único en la página, lo que significa que solo debe aparecer una vez por documento.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el contenido con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del contenido del elemento.
  • role: Define el propósito del elemento para tecnologías asistivas, como lectores de pantalla. Para `main`, se utiliza el valor `main` para identificar el contenido principal de la página.
  • aria-label: Proporciona una etiqueta accesible para el elemento, útil para describir el propósito del contenido para tecnologías asistivas.

Ejemplo


Línea 9: `id="contenidoPrincipal"` Asigna el identificador único 'contenidoPrincipal' al elemento `main`, útil para referenciarlo en CSS o JavaScript.
Línea 10: `class="contenido-principal"` La clase 'contenido-principal' se aplica al `main`, permitiendo el uso de estilos CSS definidos en la hoja de estilos.
Línea 11: `style="padding: 20px; background-color: #f4f4f4;"` Estilo en línea que establece el relleno de 20 píxeles y el color de fondo en #f4f4f4.
Línea 12: `role="main"` Define el propósito del `main` como el contenido principal de la página para tecnologías asistivas.
Línea 13: `aria-label="Contenido principal de la página"` Proporciona una descripción accesible del `main` para tecnologías asistivas.

Etiqueta <section>

<section> </section>

Se utiliza para definir una sección temática dentro de un documento. Cada sección puede contener un encabezado y otros elementos relacionados.

Especialmente útil para organizar el contenido en partes lógicas y semánticas.

Atributos

  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el contenido con JavaScript.
  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del elemento.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el elemento.
  • lang: Especifica el idioma del contenido dentro del `section`, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del `section`, como de izquierda a derecha (`ltr`) o de derecha a izquierda (`rtl`).
  • data-*: Permite almacenar datos personalizados en el `section` para su uso en JavaScript.

Ejemplo



Línea 9: `id="introduccion"`: Asigna el identificador único 'introduccion' a la sección.
Línea 10: `class="contenido-seccion"`: Aplica la clase 'contenido-seccion', que se utiliza para aplicar estilos CSS.
Línea 11: `style="border: 1px solid #ccc;"`: Estilo en línea que agrega un borde de 1 píxel alrededor de la sección.
Línea 12: `title="Introducción al tema"`: El texto emergente (tooltip) muestra "Introducción al tema" al pasar el cursor sobre la sección.
Línea 13: `lang="es"`: Indica que el contenido dentro de la sección está en español.
Línea 14: `dir="ltr"`: Define la dirección del texto como de izquierda a derecha.
Línea 15: `data-tema="ejemplo"`: Almacena el dato personalizado 'tema' con el valor "ejemplo", accesible a través de JavaScript.

Etiqueta <article>

<article> </article>

En HTML se utiliza para definir un contenido independiente y autónomo dentro de un documento, como un artículo de blog, una entrada de foro o una noticia.

Aunque la etiqueta no tiene atributos específicos, acepta varios atributos globales que se pueden utilizar para personalizar su comportamiento y apariencia.

Atributos

  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el contenido con JavaScript.
  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del elemento.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el elemento.
  • lang: Especifica el idioma del contenido dentro del `article`, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del `article`, como de izquierda a derecha (`ltr`) o de derecha a izquierda (`rtl`).
  • data-*: Permite almacenar datos personalizados en el `article` para su uso en JavaScript.

Ejemplo



Línea 9: `id="noticia1"`: Identificador único para el artículo.
Línea 10: `class="articulo"`: Clase para aplicar estilos CSS.
Línea 11: `style="border-radius: 8px;"`: Estilo en línea para bordes redondeados.
Línea 12: `title="Artículo sobre eventos recientes"`: Texto emergente (tooltip) que aparece al pasar el cursor.
Línea 13: `lang="es"`: Indica que el contenido está en español.
Línea 14: `dir="ltr"`: Define la dirección del texto como de izquierda a derecha.
Línea 15: `data-categoria="noticias"`: Dato personalizado accesible mediante JavaScript.

  • dir: Indica la dirección del texto dentro del `aside`, como de izquierda a derecha (`ltr`) o de derecha a izquierda (`rtl`).

Ejemplo

Etiqueta <h1> a <h6>

<h1> a <h6> </h1> a </h6>

Se utilizan para definir la jerarquía y la estructura del contenido en un documento.

Son fundamentales para la organización del texto y para la accesibilidad, ya que ayudan a los motores de búsqueda y a los lectores de pantalla a entender la estructura del contenido.

Atributos

  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el contenido con JavaScript.
  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del encabezado.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el encabezado.
  • lang: Especifica el idioma del contenido dentro del encabezado, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del encabezado, como de izquierda a derecha (`ltr`) o de derecha a izquierda (`rtl`).
  • data-*: Permite almacenar datos personalizados en el encabezado para su uso en JavaScript.

Ejemplo



Línea 39: `id="principal"`: Asigna un identificador único al encabezado (principal, secundario, etc.).
Línea 40: `class="encabezado"`: Aplica la clase encabezado para estilos CSS comunes a todos los encabezados.
Línea 41: `style="font-size: 2em;"`: Establece el tamaño de fuente en línea para cada encabezado.
Línea 42: `title="Encabezado Principal"`: Proporciona un texto emergente que aparece al pasar el cursor sobre el encabezado.
Línea 43: `lang="es"`: Define que el contenido de cada encabezado está en español.
Línea 44: `dir="ltr"`: Establece la dirección del texto como de izquierda a derecha (`ltr`).
Línea 45: `data-info="h1"`: Dato personalizado para cada encabezado, accesible mediante JavaScript.

Etiqueta <header>

<header> </header>

Se utiliza para definir la cabecera de un documento o de una sección. Generalmente, contiene elementos como el título, el logotipo, la navegación principal y otros elementos introductorios.

Atributos

  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular el contenido con JavaScript.
  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del contenido del `header`.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el `header`.
  • lang: Especifica el idioma del contenido dentro del `header`, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del `header`, como de izquierda a derecha (`ltr`) o de derecha a izquierda (`rtl`).
  • data-*: Permite almacenar datos personalizados en el `header` para su uso en JavaScript.
  • role: Define un rol específico para el `header` dentro de la página, mejorando la accesibilidad.
  • hidden: Oculta el contenido del `header` del renderizado visual y de las tecnologías de asistencia.

Ejemplo


Línea 9: id="paginaHeader": Asigna un identificador único `paginaHeader` al elemento `header`, útil para referenciarlo en CSS o JavaScript.
Línea 10: class="encabezado-principal": Aplica la clase `encabezado-principal`, que aplica estilos CSS como el color de fondo y el texto centrado.
Línea 11: style="border-bottom: 2px solid #333;": Aplica un estilo en línea, configurando un borde en la parte inferior del elemento `header`.
Línea 12: title="Encabezado Principal de la Página": Proporciona un texto emergente (tooltip) que aparece cuando el usuario pasa el cursor sobre el elemento `header`.
Línea 13: lang="es": Especifica que el contenido del elemento `header` está en español.
Línea 14: dir="ltr": Establece que la dirección del texto es de izquierda a derecha.
Línea 15: data-seccion="header": Almacena un dato personalizado `seccion` con el valor "header", accesible mediante JavaScript.
Línea 16: role="banner": Define el rol del elemento `header` como un banner, lo que ayuda a las tecnologías de asistencia a entender la estructura de la página.

Etiqueta <footer>

<footer> </footer>

Se utiliza para definir el pie de página de un documento o sección.

Generalmente, contiene información como derechos de autor, enlaces de contacto, información sobre el autor y otros elementos de cierre.

Atributos

  • id: Asigna un identificador único al elemento `footer`, útil para aplicar estilos específicos o manipular su contenido con JavaScript.
  • class: Asigna una o más clases al elemento `footer`, permitiendo aplicar estilos CSS o seleccionarlo con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia del `footer`, como colores, fuentes o márgenes.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el `footer`.
  • lang: Especifica el idioma del contenido dentro del `footer`, útil para accesibilidad y motores de búsqueda.
  • dir: Indica la dirección del texto dentro del `footer`, como de izquierda a derecha (`ltr`) o de derecha a izquierda (`rtl`).
  • data-*: Permite almacenar datos personalizados en el `footer` para su uso en JavaScript.
  • role: Define un rol específico para el `footer` dentro de la página, como `contentinfo` para mejorar la accesibilidad.
  • hidden: Oculta el contenido del `footer` del renderizado visual y de las tecnologías de asistencia.

Ejemplo


Línea 9: id="paginaFooter": Asigna un identificador único `paginaFooter` al elemento `footer`, útil para referenciarlo en CSS o JavaScript.
Línea 10: class="pie-de-pagina": Aplica la clase `pie-de-pagina`, que define estilos CSS como el color de fondo y la alineación del texto del `footer`.
Línea 11: style="border-top: 4px solid #444;": Establece un estilo en línea que agrega un borde en la parte superior del `footer`.
Línea 12: title="Sección de pie de página": Proporciona un texto emergente (tooltip) que aparece al pasar el cursor sobre el `footer`.
Línea 13: lang="es": Indica que el contenido dentro del `footer` está en español.
Línea 14: dir="ltr": Define la dirección del texto como de izquierda a derecha.
Línea 15: data-info="footer": Almacena un dato personalizado `footer`, accesible mediante JavaScript.
Línea 16: role="contentinfo": Define el rol del `footer` como una sección de información de contenido, mejorando la accesibilidad para tecnologías de asistencia.
