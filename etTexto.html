Etiquetas para la manipulación de texto

En HTML, las etiquetas de manipulación de texto permiten controlar la presentación y el formato del contenido textual en una página web. Estas etiquetas no solo definen la estructura del texto, sino que también aportan significado semántico, lo que ayuda a mejorar la accesibilidad y la optimización

Etiqueta de link <a>

<a> </a>

Esta etiqueta se usa para crear hiperenlaces que permiten a los usuarios navegar de una página a otra. Los enlaces pueden apuntar a páginas dentro del mismo sitio, a sitios externos o a recursos como imágenes y documentos.

Atributos

  • href: Especifica la URL del documento al que se va a enlazar. Este es el atributo principal de la etiqueta <a>.
  • target: Determina dónde se abrirá el enlace. Los valores más comunes son:
    • _self: Abre el enlace en el mismo marco (por defecto).
    • _blank: Abre el enlace en una nueva ventana o pestaña.
    • _parent: Abre el enlace en el marco superior, si el enlace está en un iframe.
    • _top: Abre el enlace en el marco completo, eliminando cualquier marco.
  • rel: Define la relación entre el documento actual y el documento al que se enlaza. Ejemplos comunes son:
    • noopener: Previene que la página de destino tenga acceso a la ventana de origen.
    • noreferrer: No envía la información de referencia al sitio de destino.
    • nofollow: Indica a los motores de búsqueda que no sigan el enlace.
  • title: Proporciona información adicional sobre el enlace que se muestra como un tooltip cuando se pasa el cursor sobre el enlace.
  • download: Indica que el enlace debe descargar el recurso en lugar de navegar a él. Puede especificar un nombre de archivo para la descarga.
  • <a href="https://www.example.com" hreflang="es">Enlace en español</a>

Ejemplo


En la línea 9: href el atributo especifica la URL del recurso al que el enlace debería llevar al usuario. En este caso, el enlace llevará al usuario a https://www.example.com. Es el atributo principal que define el destino del enlace.
En la línea 10: target indica que el enlace debe abrirse en una nueva pestaña o ventana del navegador. El valor _blank es el que se usa para abrir el enlace en una nueva pestaña.
En la línea 10: rel define la relación entre el documento actual y el recurso al que se enlaza. En este caso, noopener previene que la nueva página tenga acceso a la ventana que abrió el enlace (mejora la seguridad), y noreferrer evita que se envíe la información de referencia del enlace al sitio de destino.
En la línea 11: title proporciona un texto adicional que aparece como un tooltip cuando el usuario pasa el cursor sobre el enlace. En este caso, el tooltip mostrará "Visit Example".
En la línea 12: el dowload indica que, al hacer clic en el enlace, el navegador debe descargar el archivo en lugar de navegar hacia él. En este caso, el archivo se descargará con el nombre example.pdf. Si el enlace apuntara a un archivo, el navegador intentaría descargarlo en lugar de abrirlo.



Etiqueta <strong>

<strong> </strong>

Se emplea para definir un texto como importante, y visualmente, el texto se muestra en negrita. Esta etiqueta también es relevante para el SEO, ya que resalta contenido crucial para los motores de búsqueda.

Atributos

  • class: Permite asignar una o más clases al elemento, lo que puede ser útil para aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al elemento, que puede ser utilizado para aplicar estilos específicos, manipulación con JavaScript o para anclajes en la página.
  • title: Proporciona información adicional sobre el elemento que se muestra como un tooltip cuando el usuario pasa el cursor sobre el texto.
  • lang: Indica el idioma del contenido del elemento.
  • dir: Especifica la dirección del texto. Los valores comunes son:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.

Ejemplo


Línea 8: class="highlight": class asigna una o más clases al elemento, que se pueden utilizar para aplicar estilos CSS o seleccionar el elemento con JavaScript. En este caso, la clase highlight puede ser utilizada para aplicar estilos específicos al texto.
Línea 9: id="important-text": coloca un identificador único al elemento, que se puede utilizar para aplicar estilos específicos, manipular el elemento con JavaScript, o para crear anclajes dentro de la página. En este caso, el identificador es important-text.
Línea 10: style="color: red;": style permite aplicar estilos CSS en línea al elemento. En este caso, el texto dentro del strong se mostrará en rojo.
Línea 11: title="Important Text": proporciona un texto adicional que aparece como un tooltip cuando el usuario pasa el cursor sobre el elemento. En este caso, el tooltip mostrará "Important Text".
Línea 12: lang="en": indica el idioma del contenido del elemento. En este caso, en especifica que el texto está en inglés.
Línea 13: dir="ltr": especifica la dirección del texto. Los valores comunes son ltr (de izquierda a derecha) y rtl (de derecha a izquierda). En este caso, ltr indica que el texto debe ser leído de izquierda a derecha.

Etiqueta <small>

<small> </small>

Se utiliza para presentar texto en un tamaño menor al del texto circundante. Es ideal para notas de derechos de autor, disclaimers, o cualquier información secundaria que no sea esencial para la comprensión principal del contenido.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo la aplicación de estilos CSS o su selección en JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos, manipulación con JavaScript, o para crear anclajes en la página.
  • style: Permite aplicar estilos CSS en línea al elemento.
  • title: Proporciona información adicional que se muestra como un tooltip cuando el usuario pasa el cursor sobre el texto.
  • lang: Indica el idioma del contenido del elemento.
  • dir: Especifica la dirección del texto. Los valores comunes son:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.

Ejemplo


En la línea 9: class="note" asigna una clase llamada note al elemento "small". Esta clase puede ser utilizada para aplicar estilos CSS específicos o para seleccionar el elemento con JavaScript. En este caso, la clase note podría aplicarse para estilizar el texto de la nota al pie de página.
En la línea 10: id="reference1" asigna un identificador único reference1 al elemento "small". Este identificador permite seleccionar el elemento de manera única dentro del documento para aplicar estilos, manipularlo con JavaScript, o vincular anclajes dentro de la página.
En la línea 11: style="color: gray;" aplica estilos CSS en línea al elemento. En este caso, el texto dentro del "small" se mostrará en color gris, lo que suele utilizarse para indicar información menos destacada o aclaraciones.
En la línea 12: title="This is a footnote" proporciona un texto adicional que aparece como un tooltip cuando el usuario pasa el cursor sobre el texto. En este caso, el tooltip mostrará "This is a footnote", proporcionando información adicional sobre la nota.
En la línea 13: lang="en" indica que el contenido del elemento "small" está en inglés. Este atributo es útil para motores de búsqueda y lectores de pantalla, ayudándoles a entender y pronunciar correctamente el texto.
En la línea 14: dir="ltr" especifica la dirección del texto como "de izquierda a derecha" (ltr). Este atributo es útil para textos en lenguajes que pueden escribirse en direcciones opuestas, asegurando que se muestren correctamente en la página.

Etiqueta <cite>

<cite> </cite>

Esta etiqueta indica el título de una obra, como un libro, una película o un artículo. El texto dentro de esta etiqueta suele aparecer en cursiva, lo que ayuda a destacar las referencias y fuentes.

Atributos

  • class: Permite asignar una o más clases al elemento, lo que es útil para aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al elemento, que puede ser usado para aplicar estilos específicos, manipulación con JavaScript, o para crear anclajes en la página.
  • style: Permite aplicar estilos CSS en línea al elemento.
  • title: Proporciona información adicional que aparece como un tooltip cuando el usuario pasa el cursor sobre el texto.
  • lang: Indica el idioma del contenido del elemento.
  • dir: Especifica la dirección del texto. Los valores comunes son:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.

Ejemplo


En la línea 9: class="literature" asigna la clase literature al elemento "cite". Esta clase puede ser utilizada para aplicar estilos CSS específicos o para seleccionar el elemento con JavaScript. En este caso, literature podría aplicarse para estilizar todas las citas literarias dentro del documento.
En la línea 10: id="classic1" asigna un identificador único classic1 al elemento "cite". Este identificador permite seleccionar el elemento de manera única dentro del documento para aplicar estilos, manipularlo con JavaScript, o crear anclajes.
En la línea 11: style="font-weight: bold;" aplica estilos CSS en línea al elemento. En este caso, el texto dentro del "cite" se mostrará en negrita, lo que resalta el título de la obra citada.
En la línea 12: title="Una de las obras más importantes de la literatura española" proporciona un texto adicional que aparece como un tooltip cuando el usuario pasa el cursor sobre el texto. En este caso, el tooltip mostrará "Una de las obras más importantes de la literatura española".
En la línea 13: lang="es" indica que el contenido del elemento "cite" está en español. Esto es útil para motores de búsqueda y lectores de pantalla, ayudándoles a entender y pronunciar correctamente el texto.
En la línea 14: dir="ltr" especifica que el texto debe ser leído de izquierda a derecha, que es la dirección de lectura estándar para el idioma español

Etiqueta <sub> y <sup>

<sub> </sub> y <sup> </sup>

Estas etiquetas se usan para representar texto en subíndice y superíndice, respectivamente. Son útiles para fórmulas matemáticas, unidades químicas o notas de pie de página

Atributos

  • class: Asigna una clase específica al elemento, útil para aplicar estilos CSS.
  • id: Asigna un identificador único al elemento.
  • style: Permite aplicar estilos en línea, como cambiar el color del texto.
  • title: Proporciona un tooltip que se muestra al pasar el cursor sobre el texto.
  • lang: Indica el idioma del texto dentro del elemento.
  • dir: Especifica la dirección del texto, útil para idiomas que se leen de derecha a izquierda.
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.

Ejemplo


En la línea 10: class="subscript" asigna la clase subscript al elemento "sub". Esto podría usarse para aplicar estilos CSS específicos para subíndices.
En la línea 11: id="sub1" asigna un identificador único sub1 al elemento "sub", permitiendo seleccionarlo específicamente en el documento.
En la línea 12: style="color: red;" aplica un estilo en línea que cambia el color del texto a rojo.
En la línea 13: title="Subíndice para el número de átomos de hidrógeno" proporciona un tooltip que aparece al pasar el cursor sobre el número "2", explicando que este número es un subíndice que indica la cantidad de átomos de hidrógeno.
En la línea 14: lang="es" indica que el texto está en español.
En la línea 15: dir="ltr" especifica que el texto se lee de izquierda a derecha.
En la línea 18: class="superscript" asigna la clase superscript al elemento "sup", que puede usarse para estilizar superíndices en el documento.
En la línea 19: id="sup1" asigna un identificador único sup1 al elemento "sup", permitiendo una referencia específica dentro del documento.
En la línea 20: style="color: blue;" aplica un estilo en línea que cambia el color del texto a azul.
En la línea 21: title="Superíndice para denotar el cuadrado de la velocidad de la luz" proporciona un tooltip que aparece al pasar el cursor sobre el número "2", explicando que este número es un superíndice que denota el cuadrado de la velocidad de la luz.
En la línea 22: lang="es" indica que el texto está en español.
En la línea 23: dir="ltr" especifica que el texto se lee de izquierda a derecha.

Etiqueta <mark>

<mark> </mark>

Se utiliza para resaltar texto dentro del contenido. El texto marcado generalmente se muestra con un fondo amarillo o color de resaltado, facilitando su identificación.

Atributos

  • class: Asigna una o más clases al elemento, útil para aplicar estilos CSS.
  • id: Asigna un identificador único al elemento.
  • style: Permite aplicar estilos CSS en línea.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el elemento.
  • lang: Especifica el idioma del contenido del elemento.
  • dir: Define la dirección del texto. Los valores comunes son:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.

Ejemplo


Línea 9: Se utiliza la etiqueta Mark para resaltar la información.
En la línea 10: class="highlight" asigna la clase highlight al elemento "mark". Esto podría usarse para aplicar estilos CSS específicos para resaltados, permitiendo que el mismo estilo se aplique a múltiples elementos que compartan esta clase.
En la línea 11: id="important" asigna un identificador único important al elemento "mark". Esto permite seleccionar y manipular este elemento en particular en el documento, ya sea mediante CSS o JavaScript.
En la línea 12: style="background-color: yellow; color: black;" aplica un estilo en línea que cambia el color de fondo a amarillo y el color del texto a negro. Este atributo permite personalizar la apariencia del texto resaltado directamente en el HTML.
En la línea 13: title="Esta palabra es importante" proporciona un tooltip que aparece cuando el usuario pasa el cursor sobre el texto resaltado. Este tooltip muestra el mensaje "Esta palabra es importante", indicando la relevancia del texto dentro del contexto.
En la línea 14: lang="es" indica que el texto dentro del "mark" está en español. Esto puede ser útil para motores de búsqueda, lectores de pantalla y otros procesos que dependen del idioma del contenido.
En la línea 15: dir="ltr" especifica que el texto se lee de izquierda a derecha. Esto es útil para lenguajes que pueden ser escritos en diferentes direcciones y asegura que el texto se muestre correctamente.

Etiqueta <span>

<span> </span>

Es una etiqueta de contenedor en línea sin significado semántico propio. Se usa para aplicar estilos a una parte específica del texto mediante CSS, utilizando los atributos «class» o «id».

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipulación con JavaScript.
  • style: Permite aplicar estilos CSS en línea, directamente al elemento.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el elemento.
  • lang: Especifica el idioma del contenido del elemento.
  • dir: Define la dirección del texto. Los valores comunes son:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.
  • data-*: Permite almacenar datos personalizados en el elemento, los cuales pueden ser usados por JavaScript.

Ejemplo


En la línea 9: class="highlight" asigna la clase highlight al elemento "span". Esto permite aplicar estilos CSS específicos a todos los elementos que compartan esta clase, facilitando una personalización uniforme.
En la línea 10: id="uniqueSpan" asigna un identificador único uniqueSpan al elemento "span". Este identificador puede ser utilizado para seleccionar y manipular este elemento específico con CSS o JavaScript.
En la línea 11: style="color: blue; font-weight: bold;" aplica estilos en línea al elemento, cambiando el color del texto a azul y haciéndolo negrita. El uso del atributo style permite personalizar la apariencia del texto directamente en el HTML.
En la línea 12: title="Texto resaltado en azul" proporciona un tooltip que aparece cuando el usuario pasa el cursor sobre el texto. Este tooltip muestra el mensaje "Texto resaltado en azul", ofreciendo información adicional sobre el contenido.
En la línea 13: lang="es" indica que el texto dentro del "span" está en español. Esto es útil para motores de búsqueda y lectores de pantalla que pueden utilizar esta información para procesar mejor el contenido.
En la línea 14: dir="ltr" especifica que el texto debe leerse de izquierda a derecha, asegurando que la dirección del texto sea adecuada para la mayoría de los lenguajes que utilizan este formato.
En la línea 15: data-info="importante" almacena un dato personalizado llamado info con el valor "importante". Este atributo puede ser utilizado por JavaScript para agregar comportamientos específicos al elemento.

Etiqueta <br>

<br>

Esta etiqueta inserta un salto de línea en el texto. Es útil cuando se necesita dividir texto en varias líneas sin iniciar un nuevo párrafo.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipulación con JavaScript.
  • style: Permite aplicar estilos CSS en línea, aunque el uso de estilos en <br> es raro debido a su naturaleza de solo salto de línea.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el elemento, aunque no es común en <br>.
  • lang: Especifica el idioma del contenido del elemento, pero en el caso de <br>, su uso es raro porque <br> no contiene contenido textual.
  • dir: Define la dirección del texto. Los valores comunes son:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.
    Este atributo es raramente usado con <br>.

Ejemplo


En la línea 8: "p" es el inicio de un párrafo en el que se incluirá el texto que será interrumpido por la etiqueta "br".
En la línea 9: class="line-break" asigna la clase line-break al elemento "br". Aunque no es común aplicar estilos a un "br", esto puede ser útil para identificar y seleccionar todos los saltos de línea en un documento si es necesario.
En la línea 10: id="uniqueBreak" asigna un identificador único uniqueBreak al elemento "br". Esto permite hacer referencia a este salto de línea específico en CSS o JavaScript, aunque su uso es raro en el contexto de un "br".
En la línea 11: style="margin-bottom: 20px;" aplica un margen inferior de 20 píxeles al salto de línea, separando visualmente el texto antes y después del "br". Aunque no es común aplicar estilos directamente a "br", esto puede ser útil si necesitas ajustar el espaciado entre líneas de texto.
En la línea 12: title="Salto de línea" proporciona un tooltip que aparece cuando se pasa el cursor sobre el salto de línea. Aunque no es común que los usuarios pasen el cursor sobre un "br", este atributo está disponible si se requiere.
En la línea 13: lang="es" indica que el texto asociado está en español. Este atributo es más relevante para el contenido textual, pero puede ser utilizado con "br" como parte de una convención de etiquetado.
En la línea 14: dir="ltr" especifica que la dirección del texto es de izquierda a derecha. Este atributo se hereda del contenedor padre, pero se puede especificar en "br" si es necesario.
