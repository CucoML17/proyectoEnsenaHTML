Etiquetas para la creación de tablas y contenedores

Las etiquetas HTML para tablas son esenciales para organizar y presentar datos estructurados en forma de filas y columnas dentro de una página web. Estas etiquetas permiten a los desarrolladores crear tablas que no solo son funcionales, sino también accesibles y fáciles de entender para los usuarios y los motores de búsqueda.

Etiqueta <table>

<table> </table>

Estas son las etiquetas de apertura y cierre para una tabla en HTML. Todos los elementos de la tabla, incluyendo filas, celdas y encabezados, deben estar contenidos dentro de estas dos etiquetas.

Se utiliza para definir el inicio y el fin de la tabla, creando un contenedor para los datos tabulares.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS o seleccionar el elemento con JavaScript.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular la tabla con JavaScript.
  • style: Permite aplicar estilos CSS en línea para personalizar la apariencia de la tabla.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre la tabla, ofreciendo información adicional.
  • summary: Proporciona una descripción breve del propósito y la estructura de la tabla para usuarios de lectores de pantalla.
  • border: Controla el grosor del borde de la tabla.
  • cellpadding: Define el espacio entre el borde de la celda y el contenido de la celda.
  • cellspacing: Define el espacio entre las celdas de la tabla.
  • width: Establece el ancho de la tabla.
  • align: Controla la alineación de la tabla en la página.

Etiqueta <caption>

<caption> </caption>

Esta etiqueta se usa para proporcionar un título a la tabla. El título se muestra generalmente encima de la tabla y ayuda a describir el propósito o contenido de la tabla.

Coloca la etiqueta "caption" inmediatamente después de "table" para que el título aparezca correctamente sobre la tabla.

Atributos

  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS específicos a la leyenda de la tabla.
  • id: Asigna un identificador único al elemento, útil para aplicar estilos específicos o manipular la leyenda con JavaScript.
  • style: Permite aplicar estilos CSS en línea directamente a la leyenda de la tabla.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre la leyenda, ofreciendo información adicional.
  • lang: Especifica el idioma de la leyenda, ayudando a los navegadores y herramientas de accesibilidad a interpretar el texto en el idioma correcto.
  • dir: Define la dirección del texto en la leyenda, que puede ser:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.
  • accesskey: Define una tecla de acceso rápido que puede ser utilizada para enfocar o activar la leyenda.
  • tabindex: Establece el orden de tabulación del elemento, permitiendo que sea navegable mediante el teclado.
  • role: Define un papel (rol) accesible para la leyenda, proporcionando información adicional a las tecnologías de asistencia sobre el propósito del elemento.
  • aria-label: Proporciona una etiqueta accesible que describe el propósito del elemento para usuarios de tecnologías de asistencia.
  • aria-labelledby: Referencia el ID de otro elemento que proporciona una etiqueta accesible para la leyenda, ayudando a describir su propósito.

Ejemplo

Línea 9: id="salesCaption": Asigna el identificador único salesCaption a la leyenda, útil para referenciarla en CSS o JavaScript.
Línea 10: style="color: blue;": Aplica un color de texto azul a la leyenda mediante estilos en línea.
Línea 11: title="Resumen de ventas": Proporciona un tooltip que muestra "Resumen de ventas" cuando el usuario pasa el cursor sobre la leyenda.
Línea 12: Ventas Trimestrales por Región, Texto visible de la leyenda que proporciona una descripción sobre el contenido de la tabla. En este caso, describe que la tabla muestra las ventas trimestrales por región.

Etiqueta <colgroup>

<colgroup> </colgroup>

La etiqueta "colgroup" se utiliza para agrupar una o más columnas de una tabla y aplicarles atributos o estilos en común. Esto puede ser útil para aplicar un formato específico a varias columnas simultáneamente.

Se coloca dentro de "table", antes de las etiquetas "thead", "tbody", o "tfoot", y puede contener múltiples etiquetas "col".

Atributos

  • class: Asigna una o más clases al grupo de columnas, permitiendo aplicar estilos CSS específicos a esas columnas.
  • id: Asigna un identificador único al grupo de columnas, útil para referenciarlo en CSS o JavaScript.
  • style: Permite aplicar estilos CSS en línea directamente al grupo de columnas.
  • span: Define el número de columnas que el grupo debe abarcar. Este atributo es opcional y si no se especifica, el grupo abarca todas las columnas dentro de él.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el grupo de columnas, ofreciendo información adicional.
  • dir: Define la dirección del texto en el grupo de columnas, que puede ser:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.
  • lang: Especifica el idioma del contenido en el grupo de columnas, ayudando a los navegadores y herramientas de accesibilidad a interpretar el texto en el idioma correcto.
  • role: Define un papel (rol) accesible para el grupo de columnas, proporcionando información adicional a las tecnologías de asistencia sobre el propósito del grupo.

Etiqueta <th>

<th> </th>

La etiqueta "th" se usa para definir una celda de encabezado en una fila de la tabla. Las celdas de encabezado suelen estar en negrita y centradas, y sirven para describir el contenido de las columnas o filas; se coloca dentro de una fila "tr", generalmente en la sección "thead", para identificar los encabezados de las columnas o filas

Atributos

  • colspan: Define el número de columnas que una celda de encabezado debe abarcar horizontalmente.
  • rowspan: Define el número de filas que una celda de encabezado debe abarcar verticalmente.
  • headers: Especifica uno o más identificadores de celdas de encabezado (id) que están relacionados con la celda de encabezado actual. Ayuda a proporcionar contexto adicional para tecnologías de asistencia.
  • scope: Define el alcance del encabezado. Puede ser:
    • col: Para una columna.
    • row: Para una fila.
    • colgroup: Para un grupo de columnas.
    • rowgroup: Para un grupo de filas.
  • class: Asigna una o más clases al elemento, permitiendo aplicar estilos CSS específicos.
  • id: Asigna un identificador único al elemento, útil para referencia en CSS o JavaScript.
  • style: Permite aplicar estilos CSS en línea directamente al elemento.
  • title: Proporciona un texto emergente (tooltip) que aparece cuando se pasa el cursor sobre el elemento.
  • lang: Especifica el idioma del contenido del elemento, ayudando a los navegadores y herramientas de accesibilidad a interpretar el texto correctamente.
  • dir: Define la dirección del texto dentro del elemento, que puede ser:
    • ltr: De izquierda a derecha.
    • rtl: De derecha a izquierda.
  • role: Define un papel (rol) accesible para el elemento, proporcionando información adicional a las tecnologías de asistencia sobre el propósito del elemento.
  • aria-label: Proporciona una etiqueta accesible que describe el propósito del elemento para usuarios de tecnologías de asistencia.
  • aria-labelledby: Referencia el ID de otro elemento que proporciona una etiqueta accesible para el <th>, ayudando a describir su propósito.
  • aria-describedby: Referencia el ID de otro elemento que proporciona una descripción accesible del <th>.
